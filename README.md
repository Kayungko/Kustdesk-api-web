# 安装步骤

## 🚀 项目特色

- ✅ **完善的中文本地化** - 所有界面和提示信息都支持中文
- ✅ **优化的用户管理** - 支持账户有效期和设备数量限制管理
- ✅ **改进的系统配置** - 更直观的管理界面和配置选项
- ✅ **修复了多个前端Bug** - 提升用户体验和系统稳定性

## 📋 主要功能

- 用户管理和权限控制
- 设备管理和监控
- 地址簿和群组管理
- 系统配置和参数调整
- 完整的国际化支持

```shell
git clone https://github.com/lejianwen/rustdesk-api-web
cd rustdesk-api-web   
npm install

// 本地开发
npm run dev

// 打包
npm run build

```
